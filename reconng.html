üìå What is ReconNG?

Recon-ng is a full-featured web reconnaissance framework written in Python. It provides a powerful environment similar to Metasploit but focused on information gathering. With an interactive command-line interface, built-in modules, and automation capabilities, it's a go-to tool for OSINT gathering in bug bounty, red teaming, and pentesting engagements.

üß© Key Features

  • Modular architecture with plug-and-play recon modules
  • API key management for integrations like Shodan, Google, and more
  • Built-in database for storing results (SQLite)
  • Scriptable, supports CLI automation and chaining
  • Supports exporting data (CSV, JSON, XML)

üöÄ Getting Started

Launch ReconNG with:

Basic commands:

  • modules search ‚Äì Search for available modules
  • modules load recon/domains-hosts/google_site_web ‚Äì Load a module
  • info ‚Äì View module information
  • options set SOURCE example.com ‚Äì Set target/source
  • run ‚Äì Execute the module

üåê Real-World Use Case

Suppose you're tasked with recon on example.com. Here's a practical ReconNG workflow:

# Launch ReconNG
./recon-ng

# Load a subdomain enumeration module
modules load recon/domains-hosts/bing_domain_web
options set SOURCE example.com
run

# Use collected subdomains for DNS lookups
modules load recon/hosts-hosts/resolve
run

# Export results
show hosts
export csv /tmp/results.csv

üß† Advanced Tips & Automation

  • Use keys add to configure API keys once and reuse them across modules
  • Create workflows with workspaces to organize engagements
  • Use marketplace install to download new or custom modules
  • Run modules in sequence using custom Python scripts or batch mode
